## 🚀 快速运行演示

### 1. Boids 群体演示 (WPF可视化)

**位置**: `GPGems.Visualization/Boids/BoidsCanvas.xaml.cs`

**运行方式**:
```bash
cd GPGems.DemoApp
dotnet run
# 在Demo中选择 Boids 面板
```

**核心参数速查表**:

| 参数 | 默认值 | 调小效果 | 调大效果 | 经营游戏建议 |
|------|--------|---------|---------|------------|
| `PerceptionRange` | 25 | 只看近处，更分散 | 看更远，更凝聚 | 15-30 |
| `SeparationDist` | 10 | 挤在一起 | 互相远离 | 5-8 (游客) |
| `SeparationWeight` | 1.5 | 穿透变多 | 抖动变多 | 1.2-1.8 |
| `AlignmentWeight` | 1.0 | 方向乱，各自飞 | 全部同方向 | 0.8-1.2 |
| `CohesionWeight` | 1.0 | 群体分散 | 聚成一团 | 0.8-1.2 |
| `DesiredSpeed` | 2.0 | 慢动作 | 高速移动 | 1.5-3.0 (游客) |

**预设配置**:
```csharp
// 🦅 飞鸟群
var birdSettings = new BoidSettings {
    PerceptionRange = 30f,
    SeparationDist = 8f,
    DesiredSpeed = 5f,
    SeparationWeight = 1.2f,
    AlignmentWeight = 1.0f,
    CohesionWeight = 0.8f,
    WanderWeight = 0.5f
};

// 🐟 鱼群
var fishSettings = new BoidSettings {
    PerceptionRange = 20f,
    SeparationDist = 5f,
    DesiredSpeed = 3f,
    SeparationWeight = 1.8f,  // 鱼更密集
    AlignmentWeight = 1.2f,
    CohesionWeight = 1.2f
};

// 👨‍👩‍👧 游客户群
var visitorSettings = new BoidSettings {
    PerceptionRange = 15f,
    SeparationDist = 6f,      // 保持人际距离
    DesiredSpeed = 1.5f,      // 慢慢走
    SeparationWeight = 2.0f,  // 游客更不愿意挤
    AlignmentWeight = 0.5f,   // 不刻意对齐方向
    CohesionWeight = 0.3f     // 不刻意凝聚
};
```

## 📝 经营游戏专用算法示例代码

### 🔹 员工AI (行为树 + 寻路)

```csharp
// GPGems验证通过后，Lua移植参考
var employeeBT = new BehaviorTreeBuilder()
    .Selector("Root")
        // 优先级1: 紧急任务
        .Sequence("紧急任务")
            .Condition("有紧急任务")
            .Action("前往任务地点")  // A*寻路
            .Action("执行任务")
        .End()
        // 优先级2: 普通服务
        .Sequence("服务游客")
            .Condition("附近有需求")
            .Action("走向游客")  // A* + ORCA避障
            .Action("提供服务")
        .End()
        // 优先级3: 巡逻/休息
        .Sequence("巡逻")
            .Action("随机巡逻点")  // 流场寻路
            .Action("移动到巡逻点")
        .End()
    .End()
    .Build();
```

### 🔹 高密度人流出口 (流场 + ORCA)

```csharp
// GPGems验证：500人同时向出口疏散
var flowField = new FlowFieldPathfinder(map);
flowField.CalculateFlowField(exitX, exitY);

var orca = new ORCASimulation();
orca.AddAgents(500, spawnArea);

// 每帧更新
while (simulating) {
    // 宏观：流场提供方向
    foreach (var agent in orca.Agents) {
        var flowDir = flowField.GetDirection(agent.Position.X, agent.Position.Y);
        agent.PreferredVel = flowDir * agent.MaxSpeed;
    }
    
    // 微观：ORCA处理人与人不穿透
    orca.Update(deltaTime);
}
```
