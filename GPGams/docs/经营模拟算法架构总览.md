## 🚀 快速开始使用

### 例1：农田连块加成

```csharp
using GPGems.Core.DataStructures;

// 初始化
var farm = new GridUnionFind(100, 100);

// 玩家放置农田
farm.Activate(5, 5);
farm.Activate(5, 6);
farm.Activate(6, 5);
farm.Activate(6, 6);

// 查询加成
float bonus = farm.GetYieldBonus(5, 5);  // 2x2 → +5% = 1.05
```

### 例2：作物成熟计时

```csharp
using GPGems.Core.DataStructures;

var timer = new HierarchicalTimeWheel();

// 小麦5分钟成熟
timer.AddCropRipen(5.0f, () => {
    Console.WriteLine("小麦成熟了！");
});

// 每帧调用
timer.Tick();
```

### 例3：流场 + ORCA 游客人流

```csharp
using GPGems.AI.Pathfinding;
using GPGems.AI.CollisionAvoidance;

// 初始化
var flowField = new FlowFieldPathfinder(map);
var orca = new ORCASimulation();

// 目标是广场中央
flowField.CalculateFlowField(50, 50);

// 每帧更新
foreach (var visitor in orca.Agents)
{
    var dir = flowField.GetDirection(visitor.Position.X, visitor.Position.Y);
    visitor.PreferredVel = dir * speed;
}
orca.Update(deltaTime);
```

### 例4：美观度影响场

```csharp
using GPGems.MathPhysics.Spatial;

var beautyField = new InfluenceField(100, 100);

// 放置一座雕像（美观+0.2，半径5格
beautyField.AddBeautyBuilding(25, 25, radius: 5f, beauty: 0.2f);

// 放置工厂（噪音-0.3，半径8格
beautyField.AddNoiseSource(70, 70, radius: 8f, noise: -0.3f);

// 计算全场
beautyField.RecalculateAll();

// 查询某地块满意度
float satisfaction = beautyField.GetSatisfaction(30, 30);
```

## 🎯 庄园经营游戏 Lua 算法实现路线图

### 第一阶段（核心玩法）- 预估工作量：3-4 天
1. **GridUnionFind.lua** - 基于现有 `LuaUnionFind.lua` 封装游戏专用版本
   - 添加 `Activate(x, y)` 自动与 4 邻域合并
   - 添加 `GetYieldBonus(x, y)` 连块产量加成计算
   - 添加 `GetAllClusterSizes()` 获取所有连通块统计

2. **HierarchicalTimeWheel.lua** - 升级现有单轮时间轮
   - 实现 5 层结构（ms/sec/min/hr/day）
   - 添加 `AddCropRipen(minutes, callback)` 便捷 API
   - 添加 `DailyReset(hour, minute, callback)` 全局重置

3. **FlowFieldPathfinder.lua** - 流场寻路核心
   - 成本场 + 集成场 + 流向场
   - 支持动态障碍物更新
   - 百级 NPC 共享同一路径计算

### 第二阶段（体验提升）- 预估工作量：4-5 天
4. **ORCASimulation.lua** - 人群无穿透避障
   - 速度障碍物（VO）计算
   - 线性规划求解最优速度
   - 支持数百 Agent 实时更新

5. **InfluenceField.lua** - 多源影响叠加
   - 美观度/噪音/服务范围叠加
   - 5 种衰减函数支持
   - 增量更新优化

6. **ProductionChainScheduler.lua** - 生产链调度
   - 优先级 + 距离 + 技能匹配评分
   - 任务队列管理
   - 员工空闲/工作状态切换

### 第三阶段（特色功能）- 预估工作量：2-3 天
7. **SocialForceModel.lua** - 真实人群行为
8. **BehaviorTree.lua** - NPC 智能决策
9. **BoidsSimulation.lua** - 动物群体动画
